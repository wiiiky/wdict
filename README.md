Wdict
==================

###Introduce

使用百度翻译API实现的一个简单词典程序;

我并不打算实现多么复杂的功能，只是简单的查询翻译。

###Dependencies
		libsoup2.4-dev
		libappindicator3-dev
		libjson-glib-dev
		libgtk-3-dev
